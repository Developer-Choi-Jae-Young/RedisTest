# redis container에 접속
$ sudo docker exec -it redis-master-1 bash

# 클러스터 생성 커맨드
$ redis-cli --cluster create 127.0.0.1:7001 127.0.0.1:7002 127.0.0.1:7003 --cluster-replicas 1

# slave 추가
redis-cli --cluster add-node 127.0.0.1:7004 127.0.0.1:7001 --cluster-slave
redis-cli --cluster add-node 127.0.0.1:7005 127.0.0.1:7002 --cluster-slave
redis-cli --cluster add-node 127.0.0.1:7006 127.0.0.1:7003 --cluster-slave

-c: cluster모드로 접속한다.
-p: 접속할 node의 포트
    redis-cli -c -p 7001

node 접속 후 실행하면 cluster들의 정보를 볼 수 있다.
    cluster info

node 연결 정보를 확인한다.
    cluster nodes
